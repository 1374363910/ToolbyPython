# 樱花动漫dpl生成器（yhdmDplGenerator）

## 前言

之所以要做这个小玩意，主要还是PotPlayer太好用了（具体好在哪里，相信用过的都懂）。dpl文件是PotPlayer支持的一个存放视频链接的文件，通过爬取樱花动漫的视频链接并生成dpl文件，就可以直接在PotPlayer上看视频啦。

![image-20210217192810481](http://picbed.mooonkey.cn/md/image-20210217192810481.png)

## 使用方法

下载`dist`文件夹里的`yhdmDplGenerator.exe`文件，双击它，并输入你在樱花动漫里找到的动漫网址，比如`http://www.yhdm.io/v/3419-1.html`如图：

![image-20210217193622791](http://picbed.mooonkey.cn/md/image-20210217193622791.png)

之后同文件夹里就会生成一个dpl文件了，双击dpl，就可以在PotPlayer上观看动漫了，就不需要在网页上看了。

![image-20210217193827309](http://picbed.mooonkey.cn/md/image-20210217193827309.png)

当然，这个小工具不是很完善，有些像需要flash播放的视频链接是无法爬取的，他日在完善。
